dashCommerce 2.2.0 - Quantity Discount Patch

1 - Copy Commerce.Web to your root
2 - Drag the content of the dashCommerce-2.2.0-QtyDiscount-Patch.zip in Commerce.Web folder
3 - Allow Read Write (ASPNET or NETWORK SERVICE)
4 - Complete the wizard Install/Install.aspx
5 - Run QtyDiscount.sql file manually
6 - Run the website... Click the first add on the home page of the website and see the discount
7 - Your done.... check the apple notebook qty discount

note: discount are applied as money or percentage
         a Qty-Discount tab is available in Admin/Admin_Product_Details.aspx
